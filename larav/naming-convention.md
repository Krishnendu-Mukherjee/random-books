# Naming Convention

## Controller

> `singular` - `spasi` - `underscore` diakhiri `Controller`

- ArticleController

## Table

- posts
- project_taks
- uploaded_podcasts

> `lowercase` - `snake_case` - `plural`

## Table Pivot

- post_user

> `lowercase` - `snake_case` - `alphabet`

## Column Table

- created_at
- username

> `lowercase - snake_case`

## Model

- User
- ForumThread

> `singular` - `no underscore` - `kapital tiap kata`

## Properti Model

- $model->updated_at
- $model->title

> `lower_case` - `snake_case` - `mengikuti aturan column`

## Method Model

- $model->getAll()
- $model->getPosts()

> `camelCase` dengan huruf pertama `lowercase`

## File Blade

- all.blade.php
- all_books.blade.php

> `snake_case`
